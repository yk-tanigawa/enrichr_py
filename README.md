# Enrichr_py: A simple module for Enrichr Python API [1]

## Install

You can simply clone this GitHub repo and install in your environment.

``` 
$ git clone git@github.com:yk-tanigawa/enrichr_py.git
$ cd enrichr_py/
$ python setup.py install
```
